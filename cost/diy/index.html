人気のDIY庭づくりプロジェクト

芝張りDIY

初級

費用目安

1万〜5万円（30㎡）

所要時間

1〜2日

必要道具

4点

必要な道具

スコップレーキ目土ローラー

手順

  1. 1雑草を除去し地面を平らにならす
  2. 2床土（目土）を均一に敷く
  3. 3芝生をブロック状にカットして敷き詰める
  4. 4目土を入れてローラーで転圧
  5. 5たっぷり水やりをして根付かせる

ポイント：高麗芝なら比較的失敗が少ない。傾斜地はプロに依頼推奨。

砂利敷きDIY

初級

費用目安

5,000〜3万円（10㎡）

所要時間

半日〜1日

必要道具

4点

必要な道具

スコップ熊手防草シート砂利

手順

  1. 1雑草を除去して地面を整える
  2. 2防草シートを敷いてピンで固定
  3. 3砂利を均一に3〜5cm厚さに敷く
  4. 4砂利の量は1㎡あたり60〜80kgが目安

ポイント：防草シートを必ず敷くことで雑草対策になる。砂利の種類（玉砂利・砕石等）で雰囲気が変わる。

花壇の縁取りDIY

初級

費用目安

3,000〜2万円

所要時間

半日

必要道具

3点

必要な道具

スコップ煉瓦またはレンガ調ブロック砂

手順

  1. 1花壇の形をロープで決める
  2. 2溝を掘って砂を入れる
  3. 3レンガ・石を並べて縁を作る
  4. 4内側に腐葉土・培養土を入れる

ポイント：水糸を使うと直線・曲線が美しく仕上がる。

ウッドデッキDIY

中級

費用目安

10〜30万円

所要時間

2〜5日

必要道具

4点

必要な道具

電動ドライバー丸ノコ水平器防腐材

手順

  1. 1設計図を作成してサイズを決める
  2. 2基礎（束石）を設置して水平を確認
  3. 3根太・大引きを組んで床を構成
  4. 4デッキ材を並べてビスで固定
  5. 5防腐塗料を全面に塗装して完成

ポイント：ハードウッド（イペ・ウリン）は耐久性が高いが加工が難しい。初心者はSPF材（防腐処理済み）がおすすめ。
